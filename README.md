# Сборник РГР
Этот проект представляет собой сборник двух работ РГР, которые можно запустить и просмотреть одним файлом. 

Проект легко масштабируем, ведь в него можно без проблем добавить и другие РГР. Достаточно просто закинуть работу в папку projects, предварительно переведя её в нужную форму. 
> [!NOTE]
> Важно, чтобы РГР были реализованы с использованием библиотеки freeglut.

Переключение между проектами осуществляется с помощью клавиш N (следующий) и P (предыдущий).
## РГР, которые уже лежат в папке projects
> По экрану движется вращающийся куб, изображаемый в виде граней. Вариант: управление вращением с клавиатуры.
